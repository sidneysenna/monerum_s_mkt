# Skill: plan-feature

Use para planejar uma feature sem implementar.

## Procedimento

1. Descrever objetivo e limites.
2. Identificar modulo afetado.
3. Separar dominio, aplicacao, infraestrutura e scripts.
4. Listar dados lidos e dados gravados.
5. Verificar risco de uso indevido do schema `public`.
6. Definir validacoes e bloqueios de seguranca.
7. Definir estrategia de testes.
8. Registrar o que fica fora do escopo.
