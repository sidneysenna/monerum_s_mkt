# Skill: template-rendering

Use para desenhar renderizacao de templates.

## Procedimento

1. Exigir versoes HTML e TXT.
2. Definir variaveis aceitas pelo template.
3. Validar variaveis obrigatorias.
4. Escapar conteudo dinamico.
5. Gerar preview.
6. Registrar versao do template usada pela campanha.
7. Criar testes de renderizacao.

Nao implementar motor de templates nesta fase.
