# Skill: script-design

Use para desenhar scripts seguros.

## Procedimento

1. Definir nome do script e comando pnpm futuro.
2. Definir entradas obrigatorias e filtros.
3. Validar ambiente antes da execucao.
4. Implementar modo preview e dry-run em fase futura.
5. Exigir confirmacao para acoes reais.
6. Bloquear execucao perigosa em local/dev.
7. Exibir resumo antes da acao.
8. Registrar logs sem segredos.
9. Definir limites de lote e idempotencia.
