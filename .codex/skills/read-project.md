# Skill: read-project

Use para iniciar qualquer trabalho no projeto.

## Procedimento

1. Ler `AGENTS.md`.
2. Ler `README.md`.
3. Ler `.codex/config.toml`.
4. Ler regras em `.codex/rules/`.
5. Confirmar que o modo atual permite a acao solicitada.
6. Se a tarefa envolver banco, ler `.codex/instructions/database.md`.
7. Se a tarefa envolver e-mail, ler `.codex/instructions/mailgun.md`.
8. Se a tarefa envolver scripts, ler `.codex/instructions/scripts.md`.

Nao iniciar implementacao sem confirmar que a fase atual permite.
