# Prompt: revisar seguranca de scripts

Revise a proposta ou alteracao com foco em scripts operacionais.

Verifique:

- validacao de ambiente;
- filtros explicitos;
- preview e dry-run;
- confirmacao para envio real;
- limite de lote;
- idempotencia;
- logs sem segredos;
- bloqueio contra envio acidental para toda a base.
